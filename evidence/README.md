# Evidence Policy

This directory is for public-safe evidence only.

Recommended evidence:

- redacted screenshots of the public Mattermost timeline
- architecture diagram exports
- cost budget screenshot with personal details cropped
- Container Apps status table
- commit hashes for the main implementation and citizen image workflow

Do not commit:

- raw Mattermost exports
- database dumps
- Terraform state
- API tokens
- billing profile pages with address or phone number
- quota request screenshots containing private account data

For private evidence, use the private operations repository instead.
